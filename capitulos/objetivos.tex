\section{Objetivo General}

Diseñar e implementar una arquitectura tecnológica basada en servicios cloud que permita integrar, procesar y visualizar los datos financieros de la organización de forma automatizada, garantizando la disponibilidad de información confiable, consistente y accesible para la toma de decisiones.

\section{Objetivos Específicos}

\begin{itemize}
    \item Identificar los requerimientos del área de Finanzas para la generación de reportes.  
    \item Diseñar la arquitectura del sistema para la integración y procesamiento de datos.  
    \item Implementar infraestructura cloud mediante instancias EC2 en AWS.  
    \item Desarrollar scripts ETL para automatizar la extracción, transformación y carga de datos.  
    \item Crear dashboards interactivos en Power BI para la visualización de la información financiera.  
    \item Integrar la infraestructura cloud con Power BI para la actualización automatizada de datos.  
    \item Validar los resultados mediante retroalimentación del área financiera.  
    \item Ajustar los procesos ETL y dashboards para cumplir con los requerimientos del área.
\end{itemize}
